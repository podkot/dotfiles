# dotfiles

while on mac, `xcode-select --install` first. then:

```bash
\curl -L https://raw.githubusercontent.com/podkot/dotfiles/master/install | bash
```

ubuntu prerequisites:

```bash
sudo apt-get install curl git vim
```
